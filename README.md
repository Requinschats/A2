### Implemented controls:

- A: Move left
- D: Move right
- W: Move up
- S: Move down
- Q: Rotate left
- E: Rotate right
- P: Render points
- L: Render lines
- T: Render triangles
- Home/fn+left-arrow: reset view
- Space bar: new random olaf position
- U: Increase olaf size
- J: Decrease olaf size
